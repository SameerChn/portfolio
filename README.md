# Sameer Kumar's Portfolio

A modern, responsive portfolio website showcasing my skills, projects, and experience as a Computer Science Engineering student passionate about MERN stack development and cloud technologies.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on mobile, tablet, and desktop devices
- **Modern UI**: Dark-themed, tech-forward aesthetic with smooth animations and transitions
- **Easy Navigation**: Sidebar navigation for quick access to all sections
- **Project Showcase**: Detailed project cards with descriptions, tech stacks, and live links
- **Skills Grid**: Organized technical skills by category with certifications
- **Contact Integration**: Multiple ways to get in touch with one-click access
- **Resume Download**: Direct download link to resume PDF

## Tech Stack

**Frontend:**
- React.js
- Next.js
- Tailwind CSS
- TypeScript

**Deployment:**
- Vercel
- Azure Blob Storage (for resume hosting)

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
\`\`\`bash
git clone https://github.com/Sameerchn/portfolio.git
cd portfolio
\`\`\`

2. Install dependencies
\`\`\`bash
npm install
\`\`\`

3. Run the development server
\`\`\`bash
npm run dev
\`\`\`

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production
\`\`\`bash
npm run build
npm start
\`\`\`
